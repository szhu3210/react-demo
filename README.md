# react-demo
toy project when learning react
一个用React实现的画线段的动画。给定一组坐标，实现连线的动画，并可以暂停、恢复、重新开始。

# demo location
动画效果可以从这里预览
https://sztroy.info/demo

# Usage
cd to the directory that contains this project folder
cd react-demo-master
npm start
点击Start按钮开始画线动画（重新开始）
点击Pause暂停动画
点击Resume继续动画
点击Start重新开始动画
